Data Flow

A filter change in the UI drives a round-trip through the API and back into a reactive Vue render.

1 User adjusts filters Warehouse · Category · Status · Time period selected in UI controls.
2 Composable updates state useFilters refs change; watchers in views trigger a reload.
3 api.js issues request axios GET to localhost:8001 with query params (warehouse, category, status, month).
4 FastAPI validates & filters Pydantic types the response; apply_filters() narrows the in-memory list.
5 JSON returned Shaped by response_model; CORS headers attached.
6 Refs store raw data Views keep raw lists in refs (allOrders, inventoryItems).
7 Computed properties derive Aggregates, totals, chart inputs — cached until deps change.
8 Template renders SVG charts, tables, and metric cards update reactively.
